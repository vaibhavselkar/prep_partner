# रसायनशास्त्र / Chemistry

- अणूची रचना: प्रोटॉन (धन, केंद्रकात), न्यूट्रॉन (उदासीन, केंद्रकात), इलेक्ट्रॉन (ऋण, कक्षेत भ्रमण) / Atomic structure: proton (+, in nucleus), neutron (neutral, in nucleus), electron (–, orbits nucleus); discovered by Goldstein, Chadwick, Thomson respectively
- अणुक्रमांक (Z) = प्रोटॉन संख्या; वस्तुमानांक (A) = प्रोटॉन + न्यूट्रॉन / Atomic number (Z) = no. of protons; Mass number (A) = protons + neutrons
- इलेक्ट्रॉन मांडणी: कवचांमध्ये जास्तीत जास्त इलेक्ट्रॉन = 2n² (K,L,M,N...) / Electron configuration: max electrons per shell = 2n² (K=2, L=8, M=18...)
- आधुनिक आवर्त सारणी: मेंडेलीव्हने अणुभाराने, मोसेलेने अणुक्रमांकाने मांडली; 18 गट, 7 आवर्तने / Modern periodic table by Moseley based on atomic number; Mendeleev's was based on atomic weight; 18 groups, 7 periods
- गटात खाली गेल्यास अणुत्रिज्या वाढते, आवर्तनात डावीकडून उजवीकडे कमी होते / Atomic radius increases down a group, decreases across a period (left to right)
- रासायनिक बंध: आयनिक बंध (इलेक्ट्रॉन स्थानांतरण, धातू+अधातू, उदा. NaCl), सहसंयोजक बंध (इलेक्ट्रॉन वाटणी, उदा. H₂O, CH₄) / Chemical bonding: ionic (electron transfer, metal+nonmetal, e.g. NaCl), covalent (electron sharing, e.g. H₂O, CH₄)
- अष्टक नियम (Octet rule): अणू 8 इलेक्ट्रॉन मिळवून स्थिर होतात (हायड्रोजनसाठी 2) / Octet rule: atoms gain stability with 8 electrons in outer shell (2 for hydrogen)
- आम्ल (Acid): H⁺ आयन देते, निळा लिटमस लाल करते, pH<7 (उदा. HCl, H₂SO₄) / Acids release H⁺, turn blue litmus red, pH<7
- आम्लारी (Base): OH⁻ आयन देते, लाल लिटमस निळा करते, pH>7 (उदा. NaOH) / Bases release OH⁻, turn red litmus blue, pH>7
- उदासिनीकरण: आम्ल + आम्लारी → लवण + पाणी (Neutralization reaction) / Acid + Base → Salt + Water
- सामान्य मीठ NaCl चा pH = 7 (उदासीन) / Common salt NaCl is neutral, pH = 7
- धातू: चकाकी, तन्यता, वाहकता, इलेक्ट्रॉन गमावतात (धनविद्युती) / Metals: lustrous, malleable, ductile, conductors, lose electrons (electropositive)
- अधातू: चकाकीहीन (ग्रॅफाइट अपवाद), ठिसूळ, इलेक्ट्रॉन घेतात (ऋणविद्युती) / Non-metals: non-lustrous (except graphite), brittle, gain electrons (electronegative)
- गंज लागणे (Rusting): Fe + O₂ + H₂O → Fe₂O₃·xH₂O; टाळण्यासाठी गॅल्वनायझेशन / Rusting prevented by galvanizing (zinc coating)
- ज्वलन अभिक्रिया: इंधन + O₂ → CO₂ + H₂O + उष्णता / Combustion: fuel + O₂ → CO₂ + H₂O + heat
