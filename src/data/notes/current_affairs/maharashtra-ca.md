# महाराष्ट्र चालू घडामोडी: राज्य GR, योजना, अर्थसंकल्प, पायाभूत सुविधा / Maharashtra Current Affairs: State GRs, Schemes, Budget, Infrastructure

- हिंदुहृदयसम्राट बाळासाहेब ठाकरे समृद्धी महामार्ग नागपूर ते मुंबई (शिर्डी, नाशिकमार्गे) जोडतो, लांबी सुमारे 701 किमी, हा भारतातील सर्वात लांब द्रुतगती मार्गांपैकी एक आहे / Samruddhi Mahamarg (Hindu Hrudaysamrat Balasaheb Thackeray Nagpur-Mumbai Super Communication Expressway) connects Nagpur to Mumbai via Shirdi & Nashik, approx. 701 km, one of India's longest expressways.
- मुंबई मेट्रो लाईन 3 (कुलाबा-वांद्रे-सीप्झ) ही पूर्णपणे भूमिगत मेट्रो मार्गिका असून तिचा टप्प्याटप्प्याने विस्तार सुरू आहे / Mumbai Metro Line 3 (Colaba-Bandra-SEEPZ) is a fully underground corridor, being opened in phases.
- मुख्यमंत्री माझी लाडकी बहीण योजना (जुलै 2024 सुरू) अंतर्गत पात्र महिलांना दरमहा रु.1500 डायरेक्ट बँक खात्यात दिले जातात / Mukhyamantri Majhi Ladki Bahin Yojana (launched July 2024) provides Rs.1500/month directly to eligible women's bank accounts.
- नमो शेतकरी महासन्मान निधी योजनेत राज्य सरकार केंद्राच्या पीएम-किसान योजनेला पूरक म्हणून शेतकऱ्यांना अतिरिक्त रु.6000/वर्ष देते / Namo Shetkari Mahasanman Nidhi Yojana provides an additional Rs.6000/year to farmers, supplementing PM-KISAN.
- राज्याचा अर्थसंकल्प दरवर्षी विधानसभेत सादर होतो, अर्थमंत्री हे सादर करतात (सध्या उपमुख्यमंत्री अजित पवार वित्त खाते सांभाळतात) / State Budget is presented annually in the Assembly by the Finance Minister (currently held by Deputy CM Ajit Pawar).
- जालना-नांदेड समृद्धी महामार्ग विस्तार व वाढवण बंदर (पालघर) हे प्रमुख पायाभूत प्रकल्प आहेत / Jalna-Nanded Samruddhi extension and Vadhavan Port (Palghar) are key infrastructure projects.
- मुंबई-पुणे द्रुतगती मार्ग हा भारतातील पहिला सहा-पदरी द्रुतगती मार्ग (उद्घाटन 2002) / Mumbai-Pune Expressway was India's first six-lane expressway (opened 2002).
- लेक लाडकी योजना मुलींसाठी आर्थिक सहाय्य देते / Lek Ladki Yojana provides financial assistance for girl children.
- महाराष्ट्राची राजधानी मुंबई व उपराजधानी नागपूर (हिवाळी अधिवेशन) / Mumbai is the capital; Nagpur is the winter-session capital.
- शासन निर्णय (GR) हे शासकीय आदेश असतात जे mahasarkar/GR पोर्टलवर प्रसिद्ध होतात / A Government Resolution (GR) is an official order published via the state GR portal.
