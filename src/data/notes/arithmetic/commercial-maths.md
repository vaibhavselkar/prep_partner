# वाणिज्यिक गणित — टक्केवारी, नफा-तोटा-सूट, सरळ व चक्रवाढ व्याज, गुणोत्तर-प्रमाण, भागीदारी / Commercial Mathematics — Percentage, Profit-Loss-Discount, Simple & Compound Interest, Ratio-Proportion, Partnership

## टक्केवारी (Percentage) — मूलभूत संकल्पना / Percentage — Basic Concepts
- टक्केवारी म्हणजे शंभरावा भाग; "प्रतिशत" (per cent) म्हणजे "प्रत्येक शंभरामागे" / Percentage means "per hundred"; "per cent" literally means "for every hundred".
- x% म्हणजे x/100; टक्केवारीला अपूर्णांकात व दशांशात रूपांतरित करता आले पाहिजे (उदा. 25% = 1/4 = 0.25) / x% equals x/100; must be able to convert percentage to fraction and decimal (e.g., 25% = 1/4 = 0.25).
- एखाद्या संख्येची x% काढण्याचे सूत्र: (संख्या × x)/100 / Formula to find x% of a number: (Number × x)/100.
- एका संख्येचे दुसऱ्या संख्येशी टक्केवारी प्रमाण: (पहिली संख्या/दुसरी संख्या) × 100 / Percentage of one number relative to another: (First number/Second number) × 100.
- मूळ संख्या शोधण्याचे सूत्र: जर x% = A, तर मूळ संख्या = (A × 100)/x / To find the original number: if x% = A, then original number = (A × 100)/x.
- संख्येत वाढ किंवा घट झाल्यास टक्केवारी बदल = (बदल/मूळ संख्या) × 100 / Percentage change = (Change/Original value) × 100.
- क्रमिक टक्केवारी वाढ/घट (successive percentage change) सूत्र: अंतिम मूल्य = मूळ मूल्य × (1 ± a/100) × (1 ± b/100) / Successive percentage change: Final value = Original × (1 ± a/100)(1 ± b/100).
- जर एखादी संख्या प्रथम x% ने वाढून नंतर x% ने कमी झाली, तर निव्वळ परिणाम नेहमी घट असतो = x²/100 % / If a value increases by x% then decreases by x%, net effect is always a decrease of x²/100 %.
- लोकसंख्या वाढ/घट सूत्र: n वर्षांनंतर लोकसंख्या = P(1 ± R/100)ⁿ (R = वार्षिक वाढ/घट दर) / Population growth/decline formula: Population after n years = P(1 ± R/100)ⁿ.
- यंत्राच्या घसाऱ्याचे (depreciation) सूत्र सुद्धा वरीलप्रमाणेच वापरले जाते, पण नेहमी वजाबाकी (−) चिन्हाने / Depreciation of machinery uses the same compound formula but with a minus sign.
- जर A, B पेक्षा x% जास्त असेल, तर B, A पेक्षा [x/(100+x)]×100 % कमी असतो / If A is x% more than B, then B is [x/(100+x)]×100 % less than A.
- जर A, B पेक्षा x% कमी असेल, तर B, A पेक्षा [x/(100−x)]×100 % जास्त असतो / If A is x% less than B, then B is [x/(100−x)]×100 % more than A.

## नफा, तोटा व सूट (Profit, Loss & Discount)
- खरेदी किंमत (Cost Price - CP), विक्री किंमत (Selling Price - SP), नफा (Profit) = SP − CP (जेव्हा SP > CP) / Cost Price (CP), Selling Price (SP); Profit = SP − CP when SP > CP.
- तोटा (Loss) = CP − SP (जेव्हा CP > SP) / Loss = CP − SP when CP > SP.
- नफा % = (नफा/CP) × 100; तोटा % = (तोटा/CP) × 100 — दोन्ही नेहमी CP वर आधारित असतात, SP वर नाही / Profit % and Loss % are always calculated on CP, never on SP.
- SP काढण्याचे सूत्र नफ्यासह: SP = CP × (100 + नफा%)/100 / SP with profit: SP = CP × (100 + Profit%)/100.
- SP काढण्याचे सूत्र तोट्यासह: SP = CP × (100 − तोटा%)/100 / SP with loss: SP = CP × (100 − Loss%)/100.
- CP काढण्याचे सूत्र: CP = SP × 100/(100 + नफा%) किंवा CP = SP × 100/(100 − तोटा%) / CP = SP × 100/(100+Profit%) or SP × 100/(100−Loss%).
- अंकित किंमत (Marked Price - MP) म्हणजे लेबलवरील छापील किंमत; सूट (Discount) ही नेहमी MP वर दिली जाते, CP वर नाही / Marked Price (MP) is the printed/labelled price; Discount is always calculated on MP, not CP.
- सूट % = (सूट रक्कम/MP) × 100; विक्री किंमत = MP − सूट रक्कम / Discount% = (Discount amount/MP)×100; SP = MP − Discount amount.
- SP = MP × (100 − सूट%)/100 / SP = MP × (100 − Discount%)/100.
- क्रमिक सूट (Successive Discounts) उदा. 20% व 10% एकत्र = एकूण एकल समतुल्य सूट: 1 − (1−0.20)(1−0.10) = 28% (30% नव्हे) / Successive discounts of 20% and 10% give a net single equivalent discount of 28%, not 30% — always multiply remaining fractions.
- दुकानदार अनेकदा जास्त MP छापून सूट देतो जेणेकरून सुद्धा नफा राहील — यासाठी MP = CP × (100+अपेक्षित नफा%)/(100−सूट%) / Shopkeepers mark up price above CP so that even after discount they earn profit; MP = CP × (100+desired profit%)/(100−discount%).
- खोटा तराजू/वजनकाटा (False weight) वापरून होणारा नफा % = (खरे वजन − दिलेले वजन)/(दिलेले वजन) × 100 / Profit % due to false weight = (True weight − Given weight)/Given weight × 100.
- जर विक्रेता एकाच किमतीत दोन वस्तू विकतो, एकावर नफा % व दुसरीवर तितकाच तोटा %, तर एकूण परिणाम नेहमी तोटा असतो = (नफा%/10)² टक्के / When two articles are sold at same SP, one at profit% and other at equal loss%, there is always a net loss of (Common%/10)².
- नफा व तोटा गुणोत्तराच्या रूपात मांडून सोडवणे (Ratio method) — विशेषतः जलद उत्तरासाठी उपयुक्त / Solving using ratio method for CP:SP is a quick technique for competitive exams.

## सरळ व्याज (Simple Interest - SI)
- मुद्दल (Principal - P), व्याजदर (Rate - R% प्रतिवर्ष), कालावधी (Time - T वर्षे) / Principal (P), Rate (R% per annum), Time (T years).
- सरळ व्याजाचे सूत्र: SI = (P × R × T)/100 / Simple Interest formula: SI = (P × R × T)/100.
- एकूण रक्कम (Amount - A) = P + SI = P(1 + RT/100) / Amount A = P + SI = P(1 + RT/100).
- सरळ व्याजात प्रत्येक वर्षी व्याज सारखेच (मुद्दलावर स्थिर) राहते / In SI, interest is the same every year (calculated only on original principal).
- SI शोधण्यासाठी मुद्दल, दर, काळ यापैकी कोणतेही दोन दिल्यास तिसरा काढता येतो (P, R किंवा T साठी सूत्र फिरवून) / Given any two of P, R, T along with SI/Amount, the third can be found by rearranging the formula.
- जर रक्कम n वर्षांत दुप्पट होत असेल, तर R = 100/n (सरळ व्याजात) / If amount doubles in n years under SI, then R = 100/n.
- वेगवेगळ्या दराने वेगवेगळ्या कालावधीसाठी दिलेल्या रकमेवर एकूण SI = बेरीज करून काढतात / Total SI for different rates over different periods = sum of individual SI amounts.

## चक्रवाढ व्याज (Compound Interest - CI)
- चक्रवाढ व्याजात प्रत्येक ठराविक काळानंतर (वार्षिक/सहामाही/तिमाही) व्याज मुद्दलात मिळवले जाते व पुढील व्याज त्या नव्या रकमेवर मोजले जाते / In CI, interest is added to principal at regular intervals and next interest is calculated on the new (increased) amount.
- वार्षिक चक्रवाढीचे सूत्र: A = P(1 + R/100)ⁿ, जेथे n = वर्षांची संख्या / Annual CI formula: A = P(1 + R/100)ⁿ, n = number of years.
- CI = A − P = P[(1 + R/100)ⁿ − 1] / CI = A − P = P[(1+R/100)ⁿ − 1].
- सहामाही चक्रवाढीसाठी: दर अर्धा (R/2) व काळ दुप्पट (2n) घेतात: A = P(1 + R/200)^(2n) / For half-yearly compounding: rate is halved, time is doubled: A = P(1 + R/200)^(2n).
- तिमाही चक्रवाढीसाठी: दर एक चतुर्थांश (R/4) व काळ चौपट (4n): A = P(1 + R/400)^(4n) / For quarterly compounding: rate is quartered, time is quadrupled: A = P(1 + R/400)^(4n).
- जर काळ भिन्नात्मक (उदा. 2½ वर्षे) असेल, तर सूत्र: A = P(1+R/100)² × (1 + (R/2)/100) / For fractional time (e.g., 2½ years): A = P(1+R/100)² × (1+(R/2)/100).
- 2 वर्षांसाठी CI व SI मधील फरक = P(R/100)² / Difference between CI and SI for 2 years = P × (R/100)².
- 3 वर्षांसाठी CI व SI मधील फरक = P(R/100)²[(300+R)/100] / Difference between CI and SI for 3 years = P(R/100)²[(300+R)/100].
- CI नेहमी SI पेक्षा जास्त (किंवा समान, पहिल्या वर्षी) असते कारण व्याजावर व्याज मिळते ("interest on interest") / CI is always ≥ SI because interest is earned on interest as well.
- जर रक्कम n वर्षांत चक्रवाढीने दुप्पट होत असेल, तर 2n वर्षांत ती चौपट होते, 3n वर्षांत आठपट होते (एकाच दराने) / If a sum doubles in n years under CI, it becomes 4 times in 2n years and 8 times in 3n years at same rate.
- CI संबंधित समस्यांमध्ये अनेकदा दिलेली माहिती वेगवेगळ्या वर्षांची रक्कम (Amount) असते, त्यावरून दर व मुद्दल काढावे लागते / CI problems often give amounts for different years, from which rate and principal must be derived by division.

## गुणोत्तर व प्रमाण (Ratio & Proportion)
- गुणोत्तर (Ratio) म्हणजे दोन समान प्रकारच्या राशींची तुलना; a:b असे लिहितात (a = पूर्वपद/antecedent, b = उत्तरपद/consequent) / Ratio compares two quantities of the same kind; written a:b (a = antecedent, b = consequent).
- गुणोत्तर नेहमी सर्वात लहान पूर्णांक स्वरूपात (सर्वात मोठा सामायिक विभाजक वापरून) मांडावे / Ratios should always be expressed in simplest form (dividing by HCF).
- गुणोत्तराला अंश-छेद दोन्हींना एकाच संख्येने गुणल्यास/भागल्यास गुणोत्तर बदलत नाही / Multiplying or dividing both terms of a ratio by the same non-zero number does not change the ratio.
- प्रमाण (Proportion): जेव्हा दोन गुणोत्तरे समान असतात तेव्हा a:b :: c:d असे लिहितात; याला "चार पदांचे प्रमाण" म्हणतात / Proportion: when two ratios are equal, written a:b::c:d; called a "four-term proportion".
- प्रमाणात मधली पदे (b, c) यांचा गुणाकार = टोकाची पदे (a, d) यांचा गुणाकार — याला "गुणाकार नियम" (cross multiplication / rule of proportion) म्हणतात: a×d = b×c / In a proportion, product of means = product of extremes (cross multiplication): a×d = b×c.
- सतत प्रमाण (Continued Proportion): a:b :: b:c असल्यास b हे a व c चे मध्यगुणोत्तर (mean proportional) असते, b² = a×c, b = √(ac) / Continued proportion: a:b::b:c; b is the mean proportional, b² = ac, b = √(ac).
- तिसरे प्रमाणिक (Third Proportional): a:b :: b:c मध्ये c हे a, b चे तिसरे प्रमाणिक असते / Third proportional: given a:b::b:c, c is the third proportional to a and b.
- चौथे प्रमाणिक (Fourth Proportional): a:b :: c:d मध्ये d हे a, b, c चे चौथे प्रमाणिक असते; d = (b×c)/a / Fourth proportional: in a:b::c:d, d is fourth proportional; d = (b×c)/a.
- व्यस्त प्रमाण (Inverse/Indirect Proportion): एक राशी वाढल्यास दुसरी त्याच प्रमाणात कमी होते (उदा. काम करणारे मजूर वाढल्यास लागणारा वेळ कमी होतो) / Inverse proportion: when one quantity increases, the other decreases proportionally (e.g., more workers, less time).
- सरळ प्रमाण (Direct Proportion): एक राशी वाढल्यास दुसरीही त्याच प्रमाणात वाढते (उदा. वस्तूंची संख्या वाढल्यास किंमत वाढते) / Direct proportion: both quantities increase or decrease together in same ratio.
- संयुक्त गुणोत्तर (Compounded Ratio): a:b आणि c:d यांचे संयुक्त गुणोत्तर = ac:bd / Compounded ratio of a:b and c:d = ac:bd.
- व्यस्त गुणोत्तर (Duplicate/Sub-duplicate ratio): a²:b² ला duplicate ratio आणि √a:√b ला sub-duplicate ratio म्हणतात / a²:b² is called duplicate ratio; √a:√b is called sub-duplicate ratio.
- एकूण रक्कम दिलेल्या गुणोत्तरात विभागताना: जर एकूण रक्कम S असून गुणोत्तर a:b:c असेल, तर पहिला हिस्सा = S × a/(a+b+c), इ. / To divide a total sum S in ratio a:b:c, first share = S × a/(a+b+c), and similarly for others.

## भागीदारी (Partnership)
- भागीदारी म्हणजे दोन किंवा अधिक व्यक्तींनी एकत्र भांडवल गुंतवून व्यवसाय चालवणे व नफा/तोटा ठराविक प्रमाणात वाटून घेणे / Partnership is a business run jointly by two or more persons who invest capital and share profit/loss in an agreed ratio.
- साधी भागीदारी (Simple Partnership): सर्व भागीदारांचे भांडवल सारख्याच कालावधीसाठी गुंतवलेले असते; नफा/तोटा भांडवलाच्या गुणोत्तरात वाटला जातो / Simple Partnership: all partners invest capital for the same time period; profit/loss divided in the ratio of capitals.
- संयुक्त/चक्रीय भागीदारी (Compound Partnership): भागीदारांचे भांडवल वेगवेगळ्या कालावधीसाठी गुंतवलेले असते; नफा वाटप हे (भांडवल × काळ) या गुणाकाराच्या गुणोत्तरात होते / Compound Partnership: capitals invested for different time periods; profit divided in ratio of (Capital × Time).
- नफा वाटपाचे मूळ सूत्र: भागीदार A चा नफा हिस्सा = एकूण नफा × (A चे भांडवल × काळ)/(सर्वांच्या भांडवल×काळ यांची बेरीज) / Basic formula: Partner A's profit share = Total profit × (A's Capital×Time)/(Sum of all Capital×Time products).
- सक्रिय भागीदार (Working Partner): जो भांडवलासोबत व्यवसायात प्रत्यक्ष काम/व्यवस्थापन करतो, त्याला अधिकचे व्यवस्थापन शुल्क (management fee/commission) दिले जाऊ शकते / Working Partner: manages the business in addition to investing capital; may receive extra remuneration/commission.
- निष्क्रिय/सुप्त भागीदार (Sleeping/Dormant Partner): केवळ भांडवल गुंतवतो, व्यवसाय व्यवस्थापनात सहभागी नसतो / Sleeping/Dormant Partner: only invests capital, does not participate in day-to-day management.
- भांडवल मध्येच काढून घेतल्यास किंवा नव्याने घातल्यास, प्रत्येक कालखंडासाठी स्वतंत्र (भांडवल×काळ) गुणाकार करून बेरीज करावी लागते / If capital is withdrawn or added mid-way, calculate (Capital×Time) separately for each period and sum them.
- तोटा झाल्यासही त्याच गुणोत्तरात (भांडवल×काळ नुसार) विभागला जातो, जोपर्यंत करारात वेगळे नमूद नसेल / Losses are also divided in the same (Capital×Time) ratio unless otherwise agreed in the partnership deed.
- भागीदारी संबंधित उदाहरणे बहुधा गुणोत्तर-प्रमाण संकल्पनेवरच आधारित असतात — त्यामुळे आधी गुणोत्तर पक्के करणे आवश्यक / Partnership problems are fundamentally an application of ratio-proportion, so strong ratio fundamentals are essential.

## लक्षात ठेवा / Key Points to Remember
- नफा-तोटा नेहमी CP वर आधारित; सूट नेहमी MP वर आधारित — ही सर्वात सामान्य चूक टाळावी / Always remember: Profit/Loss % is on CP; Discount % is on MP — this is the most common exam trap.
- क्रमिक % बदल कधीही सरळ बेरीज-वजाबाकी करून काढू नये, नेहमी (1±a/100)(1±b/100) पद्धत वापरावी / Never add/subtract successive percentages directly; always use the multiplicative (1±a/100)(1±b/100) method.
- x% वाढ नंतर x% घट (किंवा उलट) झाल्यास होणारी निव्वळ घट = x²/100 % — हे सूत्र पाठ असावे / Net decrease when increasing then decreasing by same x% = x²/100 % — memorize this shortcut.
- CI व SI पहिल्या वर्षी समान असतात; फरक दुसऱ्या वर्षापासून सुरू होतो / CI and SI are equal for the first year; the difference starts appearing from the second year onward.
- 2 वर्षांसाठी CI−SI फरक = PR²/10000; ही सूत्रे पाठ असणे वेळ वाचवते / CI−SI for 2 years = PR²/10000 — memorizing key formulas saves crucial exam time.
- गुणोत्तरात एकके (units) सारखीच असावी लागतात, अन्यथा गुणोत्तर अर्थहीन ठरते / Quantities in a ratio must be of the same unit, otherwise the ratio is meaningless.
- भागीदारीत काळ हा नेहमी महिन्यांत किंवा वर्षांत सुसंगत एककात मोजावा / In partnership problems, always keep time consistent in the same unit (months or years) throughout.
- परीक्षेत आकडेमोड जलद करण्यासाठी अपूर्णांक-टक्केवारी समान मूल्ये पाठ असावीत (उदा. 1/8=12.5%, 1/6=16.67%, 1/3=33.33%, 1/7≈14.28%) / Memorize common fraction-percentage equivalents for speed (e.g., 1/8=12.5%, 1/6=16.67%, 1/3=33.33%, 1/7≈14.28%).
- MPSC गट-क परीक्षेत हा घटक अंकगणित (Arithmetic) विभागाअंतर्गत येतो व यावर दरवर्षी हमखास प्रश्न विचारले जातात / In MPSC Group-C exams, this topic falls under the Arithmetic section and questions are asked almost every year without fail.
- सराव करताना नेहमी शॉर्टकट पद्धती (गुणोत्तर पद्धत, प्रत्यक्ष सूत्र) वापरून वेळेची बचत करावी, कारण वेळ मर्यादित असतो / Always practice using shortcut methods (ratio technique, direct formulas) to save time, as the exam has strict time constraints.
