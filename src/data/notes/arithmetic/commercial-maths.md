# व्यापारी गणित (टक्केवारी, नफा-तोटा-सूट, सरळ व चक्रवाढ व्याज, गुणोत्तर-प्रमाण, भागीदारी) / Commercial Maths (Percentage, Profit-Loss-Discount, SI & CI, Ratio-Proportion, Partnership)

- टक्केवारी म्हणजे प्रति शंभर भाग; x% = x/100. / Percentage means "per hundred"; x% = x/100.
- नफा = विक्री किंमत (SP) − खरेदी किंमत (CP); तोटा = CP − SP. / Profit = SP − CP; Loss = CP − SP.
- नफा% = (नफा/CP)×100; तोटा% = (तोटा/CP)×100 — नेहमी CP वर आधारित. / Profit%/Loss% is always calculated on CP.
- सूट (Discount) ही अंकित किंमतीवर (MP) दिली जाते: SP = MP − सूट. / Discount is given on Marked Price (MP): SP = MP − Discount.
- सूट% = (सूट/MP)×100. / Discount% = (Discount/MP)×100.
- सरळ व्याज (SI) = (P×R×T)/100, जिथे P=मुद्दल, R=दर%, T=कालावधी(वर्षे). / SI = (P×R×T)/100 where P=Principal, R=rate%, T=time in years.
- चक्रवाढ व्याज (CI) = P(1+R/100)^T − P (वार्षिक चक्रवाढ). / CI = P(1+R/100)^T − P (annual compounding).
- सहामाही चक्रवाढीसाठी दर व मुदत अर्धी करून सूत्र लावा (R/2, 2T). / For half-yearly CI, halve rate and double time (R/2, 2T).
- 2 वर्षांसाठी CI व SI मधील फरक = P×(R/100)². / Difference between CI and SI for 2 years = P(R/100)².
- a:b चे गुणोत्तर म्हणजे a/b; प्रमाण म्हणजे दोन गुणोत्तरे समान (a:b = c:d ⇒ a×d = b×c). / Ratio a:b means a/b; Proportion means two ratios equal (a:b=c:d ⇒ ad=bc).
- सतत प्रमाणात (continued proportion) मधली संख्या = √(पहिली×तिसरी). / In continued proportion, middle term = √(first×third).
- भागीदारीत नफ्याची विभागणी गुंतवलेली रक्कम × कालावधी या प्रमाणात होते. / In partnership, profit is shared in ratio of (capital invested × time).
- समान कालावधीसाठी नफा फक्त भांडवलाच्या प्रमाणात विभागला जातो. / If time period is equal, profit is shared only in ratio of capitals.
- MP काढण्याचे सूत्र: MP = SP×100/(100−सूट%). / To find MP: MP = SP×100/(100−Discount%).
- क्रमवार दोन सूट (successive discounts) a% व b% असल्यास एकूण परिणाम = a+b−(ab/100)%. / Successive discounts a% and b% give net effect = a+b−(ab/100)%.
- MPSC परीक्षेत शॉर्टकट सूत्रे पाठ करून सराव करा, वेळ वाचतो. / Memorize shortcut formulas and practice for MPSC exam to save time.
