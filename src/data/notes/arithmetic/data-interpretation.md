# आकडेवारीचे विश्लेषण (Data Interpretation)

- प्रथम आलेख/तक्त्याचे शीर्षक, अक्ष (X-Y), एकके व दंतकथा (legend) काळजीपूर्वक वाचा / First read the title, axes, units, and legend before solving any question.
- वर्तुळाकृती (Pie Chart) पूर्ण वर्तुळ = 360° = एकूण मूल्याच्या 100% असते / A full pie chart equals 360° which represents 100% of the total value.
- अंश काढण्याचे सूत्र: अंश = (त्या घटकाचे मूल्य / एकूण मूल्य) × 360° / Degrees for a component = (component value ÷ total value) × 360°.
- टक्केवारी काढण्याचे सूत्र: टक्केवारी = (अंश / 360) × 100 किंवा (भाग/एकूण) × 100 / Percentage = (degrees ÷ 360) × 100, or (part ÷ total) × 100.
- दोन घटकांमधील गुणोत्तर = त्यांच्या अंशांचे किंवा टक्केवारीचे गुणोत्तर (एकूण सारखेच असल्यास) / Ratio between two components = ratio of their degrees or percentages when total is same.
- स्तंभालेख (Bar Graph) मध्ये स्तंभाची उंची/लांबी मूल्याशी समानुपाती असते / In a bar graph, bar height/length is proportional to the value it represents.
- संयुक्त स्तंभालेख (Grouped/Stacked Bar) मध्ये प्रत्येक रंग/छटा वेगळ्या श्रेणीसाठी असतो, दंतकथा नक्की तपासा / In grouped or stacked bars, each shade denotes a different category — always verify via legend.
- रेषा आलेख (Line Graph) मधील चढता उतार कल (trend) दाखवतो; दोन बिंदूंमधील फरक = वाढ/घट / A rising or falling line shows trend; difference between two points shows increase/decrease.
- सरासरी (Average) = सर्व मूल्यांची बेरीज ÷ मूल्यांची संख्या / Average = sum of all values ÷ number of values.
- टक्केवारी वाढ/घट = [(नवीन मूल्य - जुने मूल्य) / जुने मूल्य] × 100 / Percentage increase or decrease = [(new value − old value) ÷ old value] × 100.
- सारणी (Table) मधील प्रश्नांत दिलेल्या पंक्ती व स्तंभांचे शीर्षक क्रॉस-चेक करूनच बेरीज/वजाबाकी करा / In table-based questions, cross-check row and column headers before adding or subtracting.
- गुणोत्तर व प्रमाण (Ratio-Proportion) प्रश्नांत सामायिक घटक (common factor) आधी काढा / In ratio-proportion DI questions, find the common factor first to simplify calculation.
- अंदाजे (approximation) गणनेसाठी मोठ्या संख्यांना जवळच्या पूर्ण संख्येत रूपांतरित करा, वेळ वाचतो / For quick estimation, round large numbers to nearest convenient figures to save time.
- एकूण बेरीज १००% किंवा ३६०° च्या बरोबर आहे का हे पडताळून चूक टाळा / Always verify that percentages sum to 100% (or degrees to 360°) to catch data errors.
- महाराष्ट्र लोकसेवा आयोगाच्या पेपरमध्ये शेतमाल उत्पादन, जनगणना, अर्थसंकल्प यांवर आधारित तक्ते वारंवार येतात / MPSC papers often use Maharashtra-relevant data like agricultural production, census, and budget figures for DI questions.
