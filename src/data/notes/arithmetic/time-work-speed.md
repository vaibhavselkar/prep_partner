# वेग-काळ-अंतर व काम-काळ (Time-Speed-Distance & Time-Work)

- वेग = अंतर/काळ; अंतर = वेग × काळ; काळ = अंतर/वेग / Speed = Distance/Time; Distance = Speed × Time; Time = Distance/Speed
- किमी/तास ला मी/से मध्ये बदलण्यासाठी ×5/18; मी/से ला किमी/तास साठी ×18/5 / km/h to m/s: multiply by 5/18; m/s to km/h: multiply by 18/5
- सम अंतर दोन वेगांनी कापल्यास सरासरी वेग = 2xy/(x+y) (हार्मोनिक मध्य) / For equal distances at two speeds, average speed = 2xy/(x+y) (harmonic mean, not arithmetic mean)
- रेल्वे: खांब/माणूस ओलांडताना अंतर = रेल्वेची लांबी; प्लॅटफॉर्म ओलांडताना अंतर = रेल्वे+प्लॅटफॉर्मची लांबी / Train crossing a pole/person: distance = train's length; crossing a platform: distance = train's length + platform's length
- दोन रेल्वे विरुद्ध दिशेने: सापेक्ष वेग = बेरीज; एकाच दिशेने: सापेक्ष वेग = फरक / Two trains opposite directions: relative speed = sum; same direction: relative speed = difference
- प्रवाहाच्या दिशेने वेग (downstream) = बोटीचा वेग + प्रवाहाचा वेग; विरुद्ध दिशेने (upstream) = बोटीचा वेग − प्रवाहाचा वेग / Downstream speed = boat speed + stream speed; Upstream speed = boat speed − stream speed
- बोटीचा वेग = (downstream+upstream)/2; प्रवाहाचा वेग = (downstream−upstream)/2 / Boat's speed = (downstream+upstream)/2; Stream's speed = (downstream−upstream)/2
- काम: जर एखादा A दिवसात काम पूर्ण करतो, तर 1 दिवसाचे काम = 1/A / If A completes work in 'a' days, one day's work = 1/a
- एकत्र काम करताना एकत्रित 1 दिवसाचे काम = 1/a + 1/b; एकूण दिवस = (a×b)/(a+b) / Combined one day's work = 1/a + 1/b; days together = ab/(a+b)
- पाईप भरणारा (+) व रिकामी करणारा (−) एकत्र वापरल्यास निव्वळ काम = (1/a) − (1/b) / Filling pipe (+) and emptying pipe (−) together: net work = 1/a − 1/b
- LCM पद्धत: एकूण कामाला दिवसांचा LCM घेऊन प्रत्येकाची कार्यक्षमता (efficiency) काढणे सोपे पडते / LCM method: assume total work = LCM of given days to easily find each one's efficiency
- सरासरी = सर्व संख्यांची बेरीज ÷ संख्यांची एकूण संख्या / Average = sum of all values ÷ number of values
- क्रमागत संख्यांची सरासरी = मधली संख्या (विषम संख्या असल्यास) / Average of consecutive numbers = middle number (if count is odd)
- संयुक्त सरासरी (combined average) = (n1×a1 + n2×a2)/(n1+n2) / Combined average = (n1a1 + n2a2)/(n1+n2)
- वर्तुळाकार गतीत विरुद्ध दिशेने भेट होण्यासाठी वेळ = परीघ/(वेग बेरीज); एकाच दिशेने = परीघ/(वेग फरक) / Circular track meeting: opposite directions time = circumference/(sum of speeds); same direction = circumference/(difference of speeds)
