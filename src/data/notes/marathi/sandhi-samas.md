# संधि व समास / Sandhi and Samas

- **संधि**: दोन शब्दांच्या मिलनीस जणार्‍या ध्वनीमध्ये होणारा बदल
- **संधि तीन प्रकार**: स्वरसंधि, व्यंजनसंधि, विसर्गसंधि
- **स्वरसंधि नियम**: समान स्वर मिळल्यास दीर्घ होतो (अ+अ=आ)
- **गुण संधि**: य+अ = य (राज+इव = राजीव), ए+अ = ए, ओ+अ = ओ
- **वृद्धि संधि**: आ+इ = ऐ, आ+उ = औ (महा+इंद्र = महैंद्र)
- **यण संधि**: इ+स्वर = य्, उ+स्वर = व् (देवी+अभिषेक = देव्यभिषेक)
- **व्यंजनसंधि**: शब्दांच्या मिलनीस अक्षरांत बदल होतो
- **समास**: दोन पद एकत्रित होऊन नवीन अर्थ निर्माण करतात
- **समास चार प्रकार**: तत्पुरुष, अव्ययीभाव, द्वंद्व, बहुव्रीही
- **तत्पुरुष समास**: अधिकरण (खेतातील घर = खेतघर), करण (दानेसे पात = दानेपात)
- **द्वंद्व समास**: समानार्थी पदांचा योग (पिता-माता, दिन-रात)
- **अव्ययीभाव समास**: अव्यय पद प्रथम असते (प्रतिदिन, आजीवन)
- **बहुव्रीही समास**: कोणत्याही पदातून नवीन अर्थ (चक्रपाणी = चक्र धारण करणारा)
- **परीक्षेसाठी टिप्स**: संधि-विच्छेद प्रश्न वारंवार येतात, समास ओळखण्याचे कौशल्य अत्यंत महत्वाचे
