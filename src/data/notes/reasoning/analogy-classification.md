# साम्यता व वर्गीकरण / Analogy & Classification

## मूलभूत संकल्पना / Basic Concept
- साम्यता (Analogy) म्हणजे दोन घटकांमधील परस्परसंबंध ओळखून तोच संबंध दुसऱ्या जोडीत लागू करणे / Analogy means identifying the relationship between two elements and applying the same relationship to another pair.
- वर्गीकरण (Classification) म्हणजे दिलेल्या घटकांपैकी उर्वरित घटकांपेक्षा वेगळा असणारा घटक (Odd One Out) शोधणे / Classification means finding the element that is different from the rest among the given items (Odd One Out).
- या घटकातील प्रश्न शाब्दिक (Verbal/Word) व असंख्यिक-संख्यात्मक (Numerical) अशा दोन प्रकारांत विभागले जातात / Questions in this section are divided into Verbal (Word) type and Numerical type.
- प्रश्नाचे स्वरूप सामान्यतः "A : B :: C : ?" असे चिन्हांकित असते, जिथे ':' म्हणजे 'शी संबंधित' आणि '::' म्हणजे 'त्याचप्रमाणे' / The question format is usually "A : B :: C : ?", where ':' means 'is related to' and '::' means 'in the same way as'.
- प्रथम पायरी म्हणजे पहिल्या जोडीतील नेमका संबंध (relationship) ओळखणे, नंतर तोच नियम दुसऱ्या जोडीस लावणे / The first step is identifying the exact relationship in the first pair, then applying the same rule to the second pair.

## शाब्दिक साम्यता (Word Analogy) चे प्रकार / Types of Word Analogy
- समानार्थी शब्द संबंध / Synonym relationship - उदा. Big : Large :: Small : Little / e.g. Big : Large :: Small : Little.
- विरुद्धार्थी शब्द संबंध / Antonym relationship - उदा. Hot : Cold :: Day : Night / e.g. Hot : Cold :: Day : Night.
- वस्तू व तिचा उपयोग / Object and its use - उदा. Knife : Cutting :: Pen : Writing / e.g. Knife : Cutting :: Pen : Writing.
- वस्तू व तिचा वर्ग/समूह / Object and its class or group - उदा. Rose : Flower :: Mango : Fruit / e.g. Rose : Flower :: Mango : Fruit.
- वस्तू व तिचे उत्पत्तिस्थान (Source) / Object and its source - उदा. Milk : Cow :: Honey : Bee / e.g. Milk : Cow :: Honey : Bee.
- कारण व परिणाम संबंध / Cause and effect relationship - उदा. Fire : Burn :: Ice : Freeze / e.g. Fire : Burn :: Ice : Freeze.
- कर्ता व त्याचे कार्यक्षेत्र / Worker and workplace - उदा. Doctor : Hospital :: Teacher : School / e.g. Doctor : Hospital :: Teacher : School.
- कर्ता व साधन / Worker and instrument used - उदा. Carpenter : Saw :: Barber : Razor / e.g. Carpenter : Saw :: Barber : Razor.
- प्राणी व त्याचे निवासस्थान / Animal and its habitat - उदा. Fish : Water :: Bird : Sky / e.g. Fish : Water :: Bird : Sky.
- प्राणी व त्याचे पिल्लू (Male-Female-Baby) / Animal and its offspring - उदा. Cow : Calf :: Dog : Puppy / e.g. Cow : Calf :: Dog : Puppy.
- प्राणी व त्याचा आवाज / Animal and its sound - उदा. Lion : Roar :: Dog : Bark / e.g. Lion : Roar :: Dog : Bark.
- वस्तू व तिचे एकक/मोजमाप / Object and its unit of measurement - उदा. Distance : Kilometer :: Weight : Kilogram / e.g. Distance : Kilometer :: Weight : Kilogram.
- शब्द व त्याचा समूहवाचक शब्द / Word and its collective noun - उदा. Sheep : Flock :: Cattle : Herd / e.g. Sheep : Flock :: Cattle : Herd.
- स्त्री-पुरुष संबंध (Gender relationship) - उदा. King : Queen :: Man : Woman / Gender relationship - e.g. King : Queen :: Man : Woman.
- लहान-मोठे किंवा तीव्रता संबंध (Degree relationship) - उदा. Warm : Hot :: Cool : Cold / Degree relationship - e.g. Warm : Hot :: Cool : Cold.
- वस्तू व तिचा भाग (Whole-Part relationship) - उदा. Book : Page :: Tree : Leaf / Whole-Part relationship - e.g. Book : Page :: Tree : Leaf.
- संख्या व त्याचे प्रकार (Numerical/Alphabetical order) - उदा. A : E :: I : ? (स्वरांचा क्रम) / Numerical/Alphabetical relationship - e.g. A : E :: I : ? (order of vowels).
- व्यवसाय व उत्पादन / Profession and its product - उदा. Baker : Bread :: Weaver : Cloth / e.g. Baker : Bread :: Weaver : Cloth.
- देश व राजधानी / राष्ट्रीयत्व / Country and its capital / nationality - उदा. India : New Delhi :: Japan : Tokyo / e.g. India : New Delhi :: Japan : Tokyo.
- भाषा व लिपी / Language and its script - उदा. Marathi : Devanagari :: English : Roman / e.g. Marathi : Devanagari :: English : Roman.

## अक्षर साम्यता (Letter/Alphabet Analogy) / Letter Analogy
- इंग्रजी वर्णमालेतील अक्षरांचा क्रमांक लक्षात ठेवणे आवश्यक (A=1, B=2 ... Z=26) / One must memorize the position of English alphabets (A=1, B=2 ... Z=26).
- उलट क्रमांकही लक्षात ठेवावा (Z=1, Y=2 ... A=26), म्हणजे अक्षराचा उलट क्रमांक = 27 - सरळ क्रमांक / Reverse position should also be remembered (Z=1, Y=2 ... A=26); reverse position = 27 - forward position.
- अक्षरांवर बेरीज, वजाबाकी, मध्यम अक्षर (middle letter), किंवा ठराविक अंतर (gap) यांवर आधारित संबंध विचारले जातात / Questions are based on addition, subtraction, middle letter, or fixed gap between letters.
- उदा. B : D :: F : H (प्रत्येक अक्षर 2 अक्षरांनी पुढे) / e.g. B : D :: F : H (each letter moves 2 places forward).
- CBA अशा उलट क्रमाचे गट (Reverse order groups) ओळखणे आवश्यक / One must identify groups written in reverse alphabetical order like CBA.
- स्वर (Vowels: A, E, I, O, U) व व्यंजन (Consonants) यांवर आधारित वर्गीकरण प्रश्न येतात / Classification questions are based on Vowels (A, E, I, O, U) and Consonants.

## संख्या साम्यता (Number Analogy) चे प्रकार / Types of Number Analogy
- बेरीज/वजाबाकी संबंध / Addition-Subtraction relationship - उदा. 4 : 7 :: 9 : 12 (+3 चा नियम) / e.g. 4 : 7 :: 9 : 12 (rule of +3).
- गुणाकार/भागाकार संबंध / Multiplication-Division relationship - उदा. 3 : 12 :: 5 : 20 (×4 चा नियम) / e.g. 3 : 12 :: 5 : 20 (rule of ×4).
- वर्ग (Square) संबंध - उदा. 4 : 16 :: 6 : 36 / Square relationship - e.g. 4 : 16 :: 6 : 36.
- घन (Cube) संबंध - उदा. 2 : 8 :: 3 : 27 / Cube relationship - e.g. 2 : 8 :: 3 : 27.
- वर्गमूळ/घनमूळ संबंध - उदा. 25 : 5 :: 49 : 7 / Square root-Cube root relationship - e.g. 25 : 5 :: 49 : 7.
- अंकांची बेरीज/गुणाकार संबंध (Digit sum/product) - उदा. 23 : 5 :: 41 : 5 (अंकांची बेरीज) / Digit sum/product relationship - e.g. 23 : 5 :: 41 : 5 (sum of digits).
- मूळ संख्या (Prime numbers) संबंध - उदा. 2, 3, 5, 7, 11, 13, 17, 19, 23... / Prime number relationship - e.g. 2, 3, 5, 7, 11, 13, 17, 19, 23...
- सम-विषम संख्या (Even-Odd) संबंध - उदा. 4 : 8 :: 5 : 10 (दुप्पट सम संख्या) / Even-Odd relationship - e.g. 4 : 8 :: 5 : 10 (double the even number).
- क्रमवार संख्या मालिका (Series pattern) - उदा. 2, 4, 6, 8 (सम संख्या क्रम) / Series pattern - e.g. 2, 4, 6, 8 (sequence of even numbers).
- दोन क्रियांचा मिश्र नियम (Combined operation: ×2+1 इ.) - उदा. 3 : 7 :: 5 : 11 (×2+1) / Combined operation rule (×2+1 etc.) - e.g. 3 : 7 :: 5 : 11 (×2+1).
- स्थानिक किंमत / दर्शनी किंमत (Place value/Face value) यावर आधारित प्रश्न / Questions based on place value and face value of digits.

## एका शब्दाशी विसंगत घटक शोधणे (Odd One Out - Word) / Odd One Out - Word Classification
- चार किंवा पाच पर्यायांपैकी तीन/चार एकाच गटात (समान गुणधर्म) येतात व एक वेगळा असतो, तो शोधायचा असतो / Among four or five options, three/four belong to the same group (common property) and one is different — that one must be identified.
- वर्गीकरणाचा आधार: प्रकार/जात (Category), उपयोग (Use), रंग, आकार, स्थान, कार्य इत्यादी असू शकतो / Basis of classification can be category, use, colour, shape, place, function, etc.
- उदा. Apple, Mango, Potato, Banana → Potato वेगळा (कारण बाकीचे फळ आहेत, पोटॅटो भाजी आहे) / e.g. Apple, Mango, Potato, Banana → Potato is odd one out (others are fruits, potato is a vegetable).
- उदा. Delhi, Mumbai, Kolkata, Sahara → Sahara वेगळा (बाकीची शहरे आहेत, सहारा वाळवंट आहे) / e.g. Delhi, Mumbai, Kolkata, Sahara → Sahara is odd one out (others are cities, Sahara is a desert).
- उदा. Triangle, Square, Circle, Line → Line वेगळा (बाकीचे बंद आकृती आहेत) / e.g. Triangle, Square, Circle, Line → Line is odd one out (others are closed figures).
- शब्दांच्या शेवटच्या/पहिल्या अक्षरावर, अक्षरसंख्येवर किंवा स्वर-व्यंजन रचनेवर आधारित वर्गीकरणही विचारले जाते / Classification may also be based on last/first letter, number of letters, or vowel-consonant pattern of words.

## एका संख्येशी विसंगत घटक शोधणे (Odd One Out - Number) / Odd One Out - Number Classification
- संख्या मूळ (Prime) आहे की संयुक्त (Composite) यावर आधारित वर्गीकरण / Classification based on whether the number is prime or composite.
- संख्या पूर्ण वर्ग (Perfect square) किंवा पूर्ण घन (Perfect cube) आहे का, यावर आधारित / Based on whether the number is a perfect square or a perfect cube.
- संख्येतील अंकांची बेरीज ठराविक पॅटर्नमध्ये आहे का यावर आधारित / Based on whether the sum of digits follows a specific pattern.
- संख्या ठराविक अंकाने (2,3,5,7...) विभाज्य आहे का यावर आधारित गट / Grouping based on whether the number is divisible by a specific digit (2,3,5,7...).
- उदा. 8, 27, 64, 100 → 100 वेगळा (बाकी घनसंख्या आहेत, 100 पूर्ण वर्ग आहे) / e.g. 8, 27, 64, 100 → 100 is odd one out (others are cubes, 100 is a perfect square).
- उदा. 4, 9, 16, 20 → 20 वेगळा (बाकी पूर्ण वर्ग आहेत) / e.g. 4, 9, 16, 20 → 20 is odd one out (others are perfect squares).

## सोडवण्याची पद्धत / रणनीती / Method / Strategy to Solve
- सर्वप्रथम दिलेल्या जोडीतील शब्द/संख्यांमधील नेमका संबंध स्पष्टपणे व्याख्येत मांडावा / First clearly define the exact relationship between the words/numbers in the given pair.
- संबंध एकापेक्षा जास्त प्रकारे बसत असल्यास, सर्वात विशिष्ट (specific) व सुसंगत संबंध निवडावा / If the relationship fits in more than one way, choose the most specific and consistent one.
- पर्यायांमध्ये संबंध पडताळून पाहावा (relationship should hold in both direction and category) / Verify the relationship works consistently across both pairs and category.
- वेळ वाचवण्यासाठी अक्षर-क्रमांक सारणी (A-Z = 1-26) पाठ असणे उपयुक्त ठरते / Memorizing the letter-number table (A-Z = 1-26) helps save time.
- वर्गीकरणाच्या प्रश्नात आधी समान गुणधर्म शोधावा, मग तो नसलेला घटक निवडावा / In classification questions, first find the common property, then select the element lacking it.

## लक्षात ठेवा / Key Points to Remember
- साम्यता प्रश्नांत संबंधाचा प्रकार (समानार्थी, विरुद्धार्थी, कारण-परिणाम, भाग-संपूर्ण इ.) आधी ठरवा / In analogy questions, first determine the type of relationship (synonym, antonym, cause-effect, part-whole, etc.).
- वर्गीकरण प्रश्नांत नेहमी बहुसंख्य घटकांचा समान गुणधर्म शोधा, अल्पसंख्य नव्हे / In classification, always find the common property of the majority group, not the minority.
- अक्षरमाला क्रमांक: A=1, B=2, C=3 ... Z=26 (उलट: Z=1 ... A=26) पाठ असावेत / Alphabet position numbers: A=1, B=2, C=3 ... Z=26 (reverse: Z=1 ... A=26) must be memorized.
- स्वर पाच आहेत: A, E, I, O, U; उर्वरित 21 अक्षरे व्यंजन आहेत / There are five vowels: A, E, I, O, U; remaining 21 letters are consonants.
- संख्या साम्यतेत नेहमी क्रिया (operation) ओळखा - बेरीज, वजाबाकी, गुणाकार, भागाकार, वर्ग, घन / In number analogy, always identify the operation - addition, subtraction, multiplication, division, square, cube.
- सराव प्रश्नांमध्ये वेळेचे व्यवस्थापन महत्त्वाचे - प्रत्येक प्रश्नासाठी सुमारे 20-30 सेकंद ध्येय ठेवावे / Time management is crucial in practice - aim for about 20-30 seconds per question.
- MPSC परीक्षेत हा घटक बुद्धिमत्ता चाचणी (Reasoning) विभागात येतो व दरवर्षी हमखास प्रश्न विचारले जातात / This topic falls under the Reasoning section in MPSC exams and questions are regularly asked every year.
