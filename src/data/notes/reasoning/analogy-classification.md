# साम्यता व वर्गीकरण / Analogy & Classification

- शब्द साम्यता म्हणजे दोन शब्दांमधील ठराविक नाते ओळखून तेच नाते दुसऱ्या जोडीत लागू करणे / Analogy means identifying the specific relationship between a word pair and applying the same logic to another pair.
- नाते प्रकार लक्षात ठेवा: कार्य-कर्ता (शिक्षक:शाळा), वस्तू-उपयोग (चाकू:कापणे), कारण-परिणाम (आग:धूर), समानार्थी, विरुद्धार्थी, भाग-संपूर्ण (चाक:गाडी) / Key relation types: worker-workplace, tool-use, cause-effect, synonym, antonym, part-whole.
- लिंग नाते लक्षात घ्या: नर:मादी (सिंह:सिंहीण), लहान:मोठे (पिल्लू:कुत्रा) / Note gender relations: male-female, young-adult animal pairs.
- एकक-मापन नाते तपासा: किलोग्रॅम:वस्तुमान, अंश:तापमान / Check unit-measurement relations: kg-mass, degree-temperature.
- संख्या साम्यतेत क्रिया शोधा: बेरीज, वजाबाकी, गुणाकार, भागाकार किंवा वर्ग/घन संबंध / In number analogy, find the operation: addition, subtraction, multiplication, division, or square/cube relation.
- उदा. 3:9 (वर्ग) :: 5:25, किंवा 4:16 (चौपट) पॅटर्न तपासा / Example: 3:9 (square) :: 5:25; also check "multiply by same factor" pattern.
- वर्गीकरणात (Odd One Out) आधी समान गटातील घटक शोधा, नंतर वेगळा घटक ओळखा / In classification, first find the common category, then spot the odd element.
- वर्गीकरण निकष: जात/प्रजाती, कार्य, रंग, आकार, स्थान, भाषा गट, राज्य-राजधानी / Classification criteria: category, function, color, shape, location, language group, state-capital.
- अक्षर मालिकेत स्थान क्रमांक वापरा: A=1...Z=26, उलट क्रम Z=1 पद्धतही तपासा / For letter series use position numbers A=1…Z=26; also check reverse order Z=1.
- जोडी पडताळताना दोन्ही बाजूंचा क्रम (शब्दक्रम) सारखाच असावा / Verify that word order/direction matches on both sides of the pair.
- महाराष्ट्र संदर्भ: जिल्हा:मुख्यालय, नदी:उगमस्थान, अभयारण्य:जिल्हा असे गट प्रश्नात येतात / Maharashtra-specific sets like district-headquarters, river-origin, sanctuary-district are commonly asked.
- मिश्र गटात एक घटक वेगळ्या वर्गाचा (उदा. फळ गटात भाजी) असतो, तोच उत्तर / In mixed groups, the odd item belongs to a different class (e.g., a vegetable among fruits) - that is the answer.
- बहुपर्यायी प्रश्नात प्रत्येक पर्याय तपासून सर्वात योग्य नाते निवडा, घाईने उत्तर देऊ नका / Check every option against the base relation before selecting the best fit.
- सराव: रोज 10-15 analogy व classification प्रश्न सोडवा वेळेचे बंधन ठेवून / Practice 10-15 timed questions daily to build speed and accuracy.
