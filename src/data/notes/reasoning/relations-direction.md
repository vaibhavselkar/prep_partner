# रक्त संबंध व दिशाज्ञान / Blood Relations & Direction Sense

- सर्वप्रथम कुटुंब वृक्ष (Family Tree) आकृतीने काढा, चिन्हे वापरा: पुरुषासाठी □ आणि स्त्रीसाठी ○, विवाहासाठी = आणि पिढीसाठी ठळक रेषा / Always draw a family tree diagram first using □ for male, ○ for female, = for marriage, and lines for generation.
- "माझ्या वडिलांचा मुलगा" म्हणजे स्वतः किंवा भाऊ / "My father's son" means self or brother.
- "माझ्या आईच्या एकुलत्या मुलाचा मुलगा" म्हणजे स्वतःचा मुलगा / "My mother's only son's son" means own son.
- वडील/आईच्या भावाला मामा/काका, बहिणीला मावशी/आत्या म्हणतात / Father's brother = uncle (काका), mother's brother = मामा; father's sister = आत्या, mother's sister = मावशी.
- पतीचे वडील = सासरे, पत्नीची आई = सासू; जावई व सून हे संबंधही लक्षात ठेवा / Husband's father = father-in-law, wife's mother = mother-in-law; remember son-in-law & daughter-in-law terms.
- "भावाचा/बहिणीचा मुलगा" = पुतण्या/भाचा, मुलगी = पुतणी/भाची / Brother's/sister's son = nephew, daughter = niece.
- कोडी सोडवताना "फक्त" (only), "एकुलता एक" (only one) या शब्दांकडे विशेष लक्ष द्या, अर्थ बदलतो / Pay close attention to words like "only" — they change the relationship's scope.
- दिशाज्ञानात मूळ दिशा: पूर्व, पश्चिम, उत्तर, दक्षिण; सूर्य सकाळी पूर्वेला उगवतो / Basic directions: East, West, North, South; sun rises in the East.
- उजवीकडे वळल्यास घड्याळाच्या काट्याच्या दिशेने (clockwise) ९०°, डावीकडे वळल्यास counter-clockwise ९०° / Right turn = 90° clockwise, left turn = 90° counter-clockwise.
- सकाळी सूर्याकडे तोंड केल्यास उजवा हात दक्षिणेला व डावा हात उत्तरेला असतो / Facing sunrise (East) in morning, right hand points South, left hand points North.
- सावली संध्याकाळी नेहमी पूर्वेकडे पडते, सकाळी पश्चिमेकडे पडते / Shadow falls East in evening, West in morning.
- अंतर मोजताना नेहमी सरळ रेषेतील (Pythagoras/aerial) अंतर काढा, प्रत्यक्ष चाललेले अंतर नाही / Final distance = straight-line (aerial) distance, use Pythagoras theorem if paths are perpendicular.
- सुरुवातीचा बिंदू आणि शेवटचा बिंदू यामधील दिशा ही उत्तर संबंधी विचारली जाते / Direction asked is usually from starting point to final point.
- कोडीमध्ये एकाच वेळी अनेक व्यक्ती व नाती दिल्यास चार्ट/तक्ता बनवून सोडवणे सोपे जाते / For multiple persons/relations, use a table for quick solving in MPSC exam.
