# तार्किक क्षमता: न्यायमीमांसा, विधान-निष्कर्ष, विधान-गृहीतक, बैठक व्यवस्था / Logical Reasoning: Syllogisms, Statement-Conclusion, Statement-Assumption, Seating Arrangement

- न्यायमीमांसामध्ये फक्त दिलेली विधाने सत्य माना, सामान्यज्ञान वापरू नका / In syllogisms, take only given statements as true; do not use general knowledge.
- वेन आकृती (Venn diagram) पद्धत सर्वात विश्वासार्ह, सर्व शक्य केसेस तपासा / Venn diagram method is most reliable; check all possible cases before concluding.
- "सर्व A हे B आहेत" चा व्यत्यास (converse) होत नाही, पण "काही A हे B आहेत" चा व्यत्यास होतो / Universal affirmative (All A are B) does not convert directly; particular affirmative (Some A are B) converts to Some B are A.
- दोन नकारात्मक (negative) विधानांतून कोणताही निष्कर्ष निघत नाही / Two negative premises give no valid conclusion.
- एक सकारात्मक + एक नकारात्मक विधान असल्यास निष्कर्ष नेहमी नकारात्मक असतो / One positive + one negative premise → conclusion must be negative.
- "either-or" निष्कर्ष तेव्हाच लागू होतो जेव्हा दोन्ही विधानांचे विषय (subject) समान व predicate पूरक (complementary) असतात / Either-or rule applies only when subjects match and predicates are complementary.
- गृहीतक (assumption) म्हणजे अव्यक्त, गृहीत धरलेले विधान जे युक्तिवादासाठी आवश्यक असते / An assumption is an implicit, unstated premise necessary for the argument to hold.
- निषेधन चाचणी (Negation test): गृहीतक नाकारल्यास युक्तिवाद कोसळला तर ते वैध गृहीतक / Negation test: if negating a statement breaks the argument, it is a valid assumption.
- मजबूत/कमजोर युक्तिवाद: वैयक्तिक मत, अतिरेकी शब्द (केवळ, नेहमी) टाळा / Strong argument avoids extreme words like "only", "always"; relevant and direct arguments are strong.
- बैठक व्यवस्थेत रेषीय (linear), वर्तुळाकार (circular) प्रकार असतात; तोंड आत/बाहेर स्पष्ट लिहा / Seating: linear and circular types; note whether facing centre or outward.
- वर्तुळाकार मांडणीत घड्याळाच्या दिशेने डावी बाजू उजवीकडे बदलते जर तोंड बाहेर असेल / In circular arrangement facing outward, left-right directions reverse compared to facing centre.
- "उजवीकडून तिसरा" व "डावीकडून तिसरा" गोंधळ टाळण्यासाठी नेहमी संदर्भ बिंदू निश्चित करा / Always fix a reference point to avoid left-right counting confusion.
- कोडी सोडवताना निश्चित (definite) माहिती आधी बसवा, नंतर सशर्त (conditional) माहिती / Place definite clues first, then apply conditional clues in puzzles.
- MPSC मध्ये कोडी अनेकदा वय, उंची, रँक क्रमवारीसह जोडलेली असतात, टेबल बनवून सोडवा / MPSC often combines seating with ranking (age/height); use a table/grid for clarity.
